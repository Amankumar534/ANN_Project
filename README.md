# 🚢 Titanic Survival Prediction using Machine Learning

A Machine Learning web application that predicts whether a passenger would survive the Titanic disaster based on passenger details. The application is built using **Python**, **Scikit-learn**, and **Streamlit**, and is deployed for interactive predictions.
---

## 📌 Features

- Predict passenger survival probability
- Interactive Streamlit web interface
- Data preprocessing and feature engineering
- Machine Learning model trained on the Kaggle Titanic dataset
- Fast and user-friendly prediction system

---

## 🛠️ Tech Stack

- Python
- Pandas
- NumPy
- Scikit-learn
- Streamlit
- Joblib

# 📋 Prerequisites

Before running the project, make sure you have installed:

- Python 3.11 or above
- Git
- Anaconda or Miniconda
- Streamlit
---

# 🚀 Installation Guide

## Step 1: Create a Project Folder

## Step 2: Initialize Git
```bash
git init
```

## Step 3: Clone the Repository
```bash
git clone https://github.com/Amankumar534/ANN_Project.git
```

## Step 4: Create a Conda Virtual Environment
```bash
conda create -n titanic python=3.10 -y
```
Activate the environment.

## Step 5: Install Dependencies

```bash
pip install -r requirements.txt
```

# ▶️ Run the Application

```bash
streamlit run app.py
```
